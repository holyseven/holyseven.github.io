本文简要介绍在博客中常用的 Markdown 写作技巧。

## 图片

```markdown
![图片描述](图片链接)
```

可以将图片放在 `assets/images/` 文件夹中，然后使用相对路径引用。

## 链接

```markdown
[链接文字](URL)
```

## 表格

| 功能   | 语法         |
| ------ | ------------ |
| 加粗   | `**文字**`   |
| 斜体   | `*文字*`     |
| 代码   | `` `代码` `` |

## 数学公式

如果需要数学公式支持，可以在 `_layouts/default.html` 的 `<head>` 中引入 MathJax 或 KaTeX。

---

以上就是常用的 Markdown 语法，足以满足大部分博客写作需求。
